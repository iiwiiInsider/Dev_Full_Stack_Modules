# Capstone Project 3 — Blog App

A simple in-memory blog built with Node.js, Express, and EJS. No database—data resets on restart.

Features
- Create, read, update, delete (CRUD) blog posts
- Responsive, clean UI
- Method override for PUT/DELETE via forms

Getting Started
1) Install dependencies
2) Start the server
3) Open http://localhost:3000

Notes
- Posts are kept in memory and will be lost when the server restarts.
- To enable hot reload during development, use `npm run dev`.
